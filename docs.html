Introduction

A trusted WordPress.org partner, Bluehost is the perfect place to turn ideas into businesses and build, grow, and scale your online presence. Built on open source technology, Bluehost supports and participates in a wide range of open-source projects to help push the internet to its full potential.

Security and privacy are at the core of our business practices and product offerings.

Open Source

WordPress websites offer the freedom to move your website to any hosting provider or platform. Whether you need Shared, Cloud, VPS, or Dedicated hosting.

Themes and Plugiins

Easily choose from thousands of free themes and plugins for your WordPress website. Themes help simplify the design process while plugins can add powerful features with just a few clicks.

Reliable Stability

WordPress powers over 25% of the internet. With proven performance and stability, it is trusted by millions of website users and developers worldwide.

Install CLI

Requirements

  • Windows 10, Mac OSX, Linux
  • Node.js v12 or higher

Install

Mac (Homebrew)

$ brew install bluehost-cli

NPM

$ npm install bluehost-cli

Yarn

$ yarn install bluehost-cli
